Watson Bar
Un sistema de inventario


| Qué es |


Sitema de inventario simple realizado a medida. El sofware permite realizar el seguimiento de productos entrantes y salientes, para así llevar un control detallado de stock.


| Características |


  • Visualización detallada de productos que están quedando fuera de stock
  • Detalles del cantidad disponible en cada producto
  • Posibilidad de filtrar productos por categoria, busqueda por nombre, etc
  • Sección para registrar compras y ventas
  • Posibilidad de distintos administradores en el sistema
  • Registro de transacciones (quién compró, vendió y cuándo)
  • Capacidad para de agregar distintos proveedores
  • Accesible desde cualquier lugar desde un sitio web
